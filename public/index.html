📊 Brand Intelligence Dashboards

🔍 Questions Answered

  • Which brands are most visible in recent news?
  • What is the public sentiment toward each brand?
  • Which publishers and authors cover brands the most?
  • How does sentiment vary by article title vs content?

🛠️ Tools & Data Sources

  • NewsAPI (last 30 days of mobile news)
  • Google BigQuery + Python (scheduled pipeline)
  • TextBlob Sentiment Analysis
  • Looker Studio + Vercel Deployment
